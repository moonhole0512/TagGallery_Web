# Tag Gallery Web UI

AI가 생성한 이미지의 태그를 데이터베이스에 저장하고, 이를 웹 인터페이스를 통해 효율적으로 검색하고 관리하는 프로그램입니다.

## 주요 기능

- **세련된 웹 UI**: 다크 모드를 기반으로 한 반응형 웹 갤러리.
- **무한 스크롤**: 스크롤을 내리면 다음 이미지들이 자동으로 로드되어 편리하게 탐색할 수 있습니다.
- **강력한 검색 및 필터링**:
    - 태그 키워드 검색
    - 정렬 (최신순, 오래된순, 무작위)
    - 플랫폼별 필터링 (NovelAI, StableDiffusion 등)
- **자세한 정보 제공**:
    - 갤러리 이미지 위에 플랫폼 정보 오버레이 표시.
    - 이미지 상세 보기에서 메타데이터를 긴 항목과 짧은 항목으로 자동 그룹화.
    - 긴 프롬프트 등 내용이 많은 항목 옆에는 복사 버튼 제공.
- **유연한 데이터 관리**: 이미지의 모든 메타데이터를 DB에 저장하여 향후 확장에 용이.
- **간편한 설정 및 실행**:
    - 웹 UI 내에서 이미지 경로 설정 가능.
    - `run.bat` 파일로 간편하게 서버 실행.

## 설치 방법

1.  **Python 설치**: Python 3.9 이상이 설치되어 있어야 합니다.
2.  **저장소 복제 또는 다운로드**:
    ```bash
    git clone https://github.com/moonhole0512/TagGallery_Web.git
    cd TagGallery_Web
    ```
3.  **필요 라이브러리 설치**:
    프로젝트 폴더에서 다음 명령어를 실행합니다.
    ```bash
    pip install -r requirements.txt
    ```

## 실행 방법

1.  프로젝트 폴더에 있는 `run.bat` 파일을 더블클릭하여 실행합니다.
2.  콘솔 창에 포트 번호를 입력하라는 메시지가 나타납니다. 원하는 포트 번호를 입력하고 Enter 키를 누릅니다. (입력하지 않으면 기본값 8000번으로 실행됩니다.)
3.  웹 브라우저를 열고 `http://127.0.0.1:[입력한 포트 번호]` (예: `http://127.0.0.1:8002`)로 접속합니다.

## 사용 방법

### 1. 최초 설정
- 처음 접속하면 'Settings' 창이 나타납니다.
- **Source Image Path**: 분류할 원본 이미지들이 있는 폴더 경로를 입력합니다.
- **Destination Image Path**: 분류된 이미지들이 저장될 폴더 경로를 입력합니다.
- `Save changes`를 눌러 설정을 저장하면, 웹 페이지가 새로고침됩니다.

### 2. 이미지 스캔
- 우측 상단의 `Scan` 버튼을 클릭하여 이미지 분류 및 정보 추출을 시작합니다.
- 이 작업은 백그라운드에서 진행되며, 터미널(콘솔) 창에서 진행 상황을 확인할 수 있습니다.

### 3. 갤러리 탐색 및 검색
- **탐색**: 메인 화면에서 마우스 휠을 아래로 스크롤하면 다음 이미지들이 자동으로 로드됩니다.
- **검색 및 필터링**:
    - **검색창**: 원하는 태그나 키워드를 입력하고 `Search` 버튼을 누르거나 Enter 키를 치면 검색 결과가 나타납니다.
    - **Sort**: 정렬 드롭다운 메뉴에서 `Newest`(최신순), `Oldest`(오래된순), `Random`(무작위)을 선택하여 이미지 정렬 순서를 변경할 수 있습니다. (기본값: Random)
    - **Platform**: 플랫폼 드롭다운 메뉴에서 `NovelAI`, `StableDiffusion` 등을 선택하여 특정 플랫폼에서 생성된 이미지만 필터링할 수 있습니다. (기본값: All)
    - 필터나 정렬 옵션을 변경하면 즉시 갤러리가 다시 로드됩니다.

### 4. 이미지 상세 정보 확인
- 갤러리에서 마음에 드는 이미지를 클릭하면 상세 정보 창이 나타납니다.
- 확대된 이미지와 함께 아래쪽에 모든 메타데이터 정보가 표시됩니다.
- `prompt`와 같이 내용이 긴 정보는 큰 텍스트 박스에, `seed`, `steps` 등 짧은 정보는 하단의 'Details' 섹션에 그룹화되어 표시됩니다.
- 내용이 긴 정보 옆의 `Copy` 버튼을 클릭하면 해당 내용을 쉽게 복사할 수 있습니다.
